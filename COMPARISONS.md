# Comparisons (High-Level)

Nu11VLT differs from mainstream photo apps in one fundamental way:

**Your photos never leave your device, and no third party holds encryption keys.**

---

## Feature Comparison

| Category | Typical Cloud Photo Apps | Nu11VLT |
|----------|-------------------------|---------|
| Cloud storage | Required | None |
| Zero-knowledge | No | Yes |
| Offline operation | Limited | Full |
| Dead-man's switch | No | Optional |
| Plausible deniability | No | Optional |
| Forensic-resistant deletion | No | Yes |

---

Nu11VLT is designed for sensitive local storage, not convenience, sync, or social features.
